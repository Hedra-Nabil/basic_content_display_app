To add images to this app:

1. Place your image files in this folder (assets/images/)
2. Name them: nature1.jpg and nature2.jpg
   OR update the image paths in lib/main.dart to match your image filenames
3. Supported formats: .jpg, .jpeg, .png, .gif, .webp
4. After adding images, run: flutter pub get

Example:
- assets/images/nature1.jpg
- assets/images/nature2.jpg
